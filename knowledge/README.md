# Knowledge

这个目录用于存放已经稳定、可复用的知识笔记。不要把临时想法、单次对话过程或未验证假设直接放进这里。

## 晋升标准

内容从 `sessions/` 晋升到 `knowledge/` 前，应满足至少大部分条件：

- 多次被使用，或明显会服务未来研究问题。
- 有明确来源支持。
- 表述已经稳定，不只是临时想法。
- 能归入清晰主题。
- 不与已有笔记重复。

## 笔记建议

稳定知识笔记建议包含：

- Definition：概念定义。
- Key mechanisms：关键机制。
- Evidence：证据来源和证据强度。
- Common confusions：常见误解。
- Relation to AI：与 AI 的关系，如不相关可省略。
- Open questions：仍未解决的问题。
- Sources：来源列表。

# NeuroScience Research Workspace

这是一个 **神经科学 × AI（Neuroscience × AI）研究工作区**，用于长期学习、研究、追踪前沿、整理论文、管理问题与假设。

本项目遵循 [AGENTS.md](/Users/phoenix/Assistant/NeuroScience/AGENTS.md) 中的研究助理规则：研究为主，学习服务于研究；强调证据等级、批判性判断、机制解释和结构化记录。

## 项目主页

- 本地主页：[index.html](/Users/phoenix/Assistant/NeuroScience/index.html)
- GitHub Pages：部署后使用 `https://mingxiangbian.github.io/NeuroScience/`

## 如何使用

优先把内容放到合适的位置：

- `sessions/`：重要对话和研究 session 记录。
- `knowledge/`：已经稳定、可复用的主题知识。
- `papers/`：一篇论文一份结构化笔记。
- `projects/`：较大的研究问题或长期课题。
- `sources/`：资料来源、数据库、书目、链接集合。
- `questions/`：尚未归入具体项目的开放问题和假设。
- `templates/`：记录模板。

## 记录原则

- 普通问答不自动记录。
- 形成阶段性结论、阅读论文、推进项目、维护开放问题时再创建或更新文件。
- 结论要区分证据强度和来源类型。
- 重要术语使用中文为主，并保留英文术语，方便检索。

## 推荐流程

1. 在 `sessions/` 记录重要研究对话。
2. 把反复出现、来源稳定的内容整理进 `knowledge/`。
3. 阅读论文时用 `templates/paper-note.md` 建立笔记。
4. 遇到大问题时，在 `projects/` 建立项目文件夹。
5. 用 `INDEX.md` 维护当前研究地图。
